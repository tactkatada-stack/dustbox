# 動画ファイルの置き場所

ここに、ゴミ箱の充填段階ごとの数秒動画(mp4)を置いてください。
ファイル名は `app.js` の `STAGES` で指定しているものと一致させる必要があります。

| ファイル名 | 想定する充填率 | 内容 |
|---|---|---|
| `stage-empty.mp4` | 0〜24% | 空の状態 |
| `stage-low.mp4` | 25〜49% | 少し溜まった状態 |
| `stage-mid.mp4` | 50〜74% | 半分くらい溜まった状態 |
| `stage-full.mp4` | 75〜100% | 満杯の状態 |

- 各動画は自動でループ再生されるので、数秒のループしても違和感のない映像がおすすめです
- 段階の数やしきい値を変えたい場合は `app.js` の `STAGES` 配列を編集してください
- ファイルがまだ置かれていない/読み込みに失敗した場合は、公開ページ上に「動画未設定」というプレースホルダーが表示されます
