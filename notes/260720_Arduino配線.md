# 260720_Arduino配線

HC-SR04(超音波距離センサー)をArduino Unoに配線する手順のメモ。
ブレッドボード・ジャンプワイヤを使う前提。

配線: `VCC→5V, GND→GND, Trig→9番, Echo→10番`

## 1. まず、HC-SR04の4本のピンを知る

HC-SR04には端子が4本あります(基板に文字が印字されています)。

| 表示 | 意味 |
|---|---|
| **VCC** | 電源(+) |
| **Trig** | 「測定して」という信号を送る用 |
| **Echo** | 測定結果(反射時間)が返ってくる用 |
| **GND** | 電源(−・グランド) |

## 2. ブレッドボードの仕組みを軽く理解する

ブレッドボードは穴の下で電気的に繋がっているグループがあります。

- **端(はし)にある2本の縦の帯(+と−が書いてある列)**: 縦一列がすべて繋がっている(電源用のレール)
- **真ん中の部分**: 5個ずつの穴が横方向に繋がっている(小さいグループ単位)

今回は部品点数が少ないので、真ん中の穴を使ってHC-SR04を挿し、そこからジャンプワイヤでArduinoに繋ぐ形にする。

## 3. 配線手順

1. **HC-SR04をブレッドボードの真ん中に挿す**
   4本の足を、ブレッドボード中央の溝をまたぐように挿す(4本がそれぞれ別の行に刺さればOK)
2. **ジャンプワイヤを4本用意する**
3. 以下の対応でArduinoとブレッドボードを繋ぐ

   | HC-SR04側(ブレッドボードの、VCCの足と同じ行) | Arduino側 |
   |---|---|
   | VCCの行 | **5V** |
   | GNDの行 | **GND** |
   | Trigの行 | **9番ピン**(デジタルピン) |
   | Echoの行 | **10番ピン**(デジタルピン) |

## 4. Arduino Uno側のピンの場所

Arduino Unoの基板には文字が印字されている。

- **5V** と **GND** は、基板の「POWER」と書かれているエリアにある
- **9** と **10** は、基板の「DIGITAL」と書かれているエリア(数字が並んでいる列)にある。印字通りの数字のところに挿す

## 5. 確認のポイント(重要)

- **VCCとGNDを逆に挿さないように注意**する(センサーが壊れる可能性がある)。挿す前にもう一度、HC-SR04側の印字とジャンプワイヤの行き先を見比べる
- ジャンプワイヤは同じ「行(グループ)」に挿さっていれば向きはどちらでも大丈夫
- Arduinoをまだパソコンに繋いでいない状態で配線する(繋いだまま触ると誤動作やショートのリスクがあるので、配線が終わってからUSB接続するのが安全)

## 6. 配線が終わったら

1. ArduinoをUSBでPCに接続
2. Arduino IDEで `arduino/dustbox_sensor/dustbox_sensor.ino` を開いて書き込み
3. 「ツール」→「シリアルモニタ」を開く(通信速度を **9600** に設定)
4. センサーの前で手を近づけたり離したりして、数値(距離cm)が変化すれば配線成功
